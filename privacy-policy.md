# Politique de confidentialité

Cette application Messenger est utilisée pour recevoir et traiter les messages envoyés à notre Page Facebook afin de fournir des réponses automatiques ou assistées.

## Données collectées
Nous pouvons traiter les données suivantes :
- le contenu des messages envoyés à la Page
- les identifiants techniques nécessaires au fonctionnement de Messenger
- les événements liés aux interactions de messagerie

## Utilisation des données
Ces données sont utilisées uniquement pour :
- recevoir les messages entrants
- traiter les demandes des clients
- envoyer des réponses automatiques ou assistées
- améliorer le fonctionnement du service client

## Partage des données
Les données ne sont pas revendues à des tiers.

## Conservation des données
Les données sont conservées uniquement le temps nécessaire au fonctionnement du service.
